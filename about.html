MangaDL

What is this?

A web downloader for manga hosted on Shonen Jump Plus, Sunday Web Every, and maybe others (the two use the same viewer).

How does it work?

Manga metadata and pages are loaded from their original servers through a CORS proxy (hosted on Heroku). Then, the shuffling of the pages is undone using the HTML5 Canvas API. Unshuffled pages are zipped (in parallel, as the images are being loaded/processed!) using fflate. The resulting ZIP is saved using FileSaver.js.

This means that all work happens in the client. Nothing is stored server-side.

How about support for other sites?

I may add support for other sites as I need it. In the meantime, you are more than welcome to create a PR.
